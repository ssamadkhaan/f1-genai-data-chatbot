# 🏎️ F1 GenAI Data Chatbot

An AI-powered Formula 1 chatbot built using Python, Google Gemini API, Pandas, and Streamlit.

## Features

- Natural language F1 question answering
- Gemini-powered responses
- Tool calling architecture
- Memory-enabled conversations
- Multi-tool agent system
- Streamlit web interface
- F1 driver, constructor, and race analysis

## Tech Stack

- Python
- Pandas
- Google Gemini API
- Streamlit
- Git & GitHub

## Project Structure

data/
notebook/
app.py
requirements.txt

## Sample Questions

- Tell me about Lewis Hamilton
- Tell me about Ferrari
- Show Australian Grand Prix races
- Which team does Lewis Hamilton drive for?

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
